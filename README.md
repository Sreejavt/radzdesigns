# Radz Designs - Conversion Architecture Website

## Overview
Radz Designs is a conversion architecture practice specializing in diagnosing and fixing ROI leaks in Saas and other digital funnels through Applied Psychology and systematic design principles.

## Project Structure
This website consists of five main pages:
- **Home** - Introduction to the company and core value proposition
- **Audit** - Conversion audit services, scope and pricing
- **Funnels** - Funnel Build and redesign services, scope and pricing
- **Framework** - The 14-point conversion architecture framework, scope and application
- **Contact** - Contact Form and Inquiry Submission

## Services Offered
### Conversion Audits
- Tier 1: Diagnostic Audit ($2,500-$3,000)
- Tier 2: $5K Pilot - High-Impact Page Audit + Fix Blueprint

### Funnel Design
- Full Funnel Build (from scratch)
- Funnel Redesign (optimization & re-architecture)

## The 14-Point Framework
Diagnostic system organized into three categories:
1. **Cognitive Clarity** - Five-Second Value Test, Why Now Relevance, Buyer Role Clarity, Product Differentiation, Destination vs Features, Value Visibility Gap
2. **Behavioral Flow** - Pre-Click to Landing Transition, Use Case & Aha Moment, Information Flow Architecture, CTA Architecture, Psychological Friction Points
3. **Structural Integrity** - Visual Hierarchy, Value Prioritization Flaw, Final ROI Leak Diagnosis
